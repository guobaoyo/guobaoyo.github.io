本笔记是基于弗朗索瓦•肖莱(François Chollet)的DeepLearning with Python书籍的课程笔记。

在看书中曾遇到的不会的点：

1、在P22有个准备图像数据

  • astype的作用是将数据类型转换为后面（）部分的数据类型，在这里是float类型。

  • 后面除以255是因为灰度图的范围是[0,255]，然后再除以255，这样每个值的范围都集中在[0,1]，便于计算。

  • to_categorical()函数的作用是将类别向量转化为二进制（0和1）的矩阵类型表示。

2、epoch 和iteration 和 batchsize的区别（假设整个训练集为N）

  • batchsize可大可小，如果大到N，那么这叫full batch learning

  • batchsize如果小到1，就是每次只训练一个样本，就叫做online learning

  • 所以batchsize一般会取中间值n

    iteration =N/n

    epoch：整个N个训练集全部训练过一遍后epoch增加1 ，一般不会将epoch设置为1，因为一个大型的神经网络不可能将训练集通过一遍就训练完毕，所以随着epoch的增加，模型将从欠拟合逐步转化到过拟合。

书中写的是“避免覆盖输入张量”

但是我查copy()和deepcopy()的区别：https://blog.csdn.net/u010712012/article/details/79754132 里面讲copy()是浅复制，改变原来的值那么拷贝出来的值也会改变（在上面的文章中有标签的概念讲的很清晰可以参考）。deepcopy()是深复制，拷贝出来的东西与原来的是独立的。此外我又在电脑上试了一下，在整体改变x1的值之后必须再copy一遍才会生效，详见贴图：

书中写的“避免覆盖输入张量”我觉着应该用deepcopy()更合适，这样会独立出来一个对象对其进行修改并不会对原来的x造成影响。这样才会达到注释写的“避免覆盖”的作用。

5、python中数组，元组，列表的区别：

  • 在python中主要是元组和列表这两个概念，主要区别在于元组创建之后不可变，不能再append元素，也不能pop出来元素，且只能进行索引，不能改变其值（已经写死了） 元组注意是小括号(1,2,3)

  • 列表可以改变，可以索引，可以赋值，可以append和pop元素 列表注意是中括号[1,2,3]

  • 数组主要是python中的numpy的概念，例如：

    数组和列表的区别在于：

    • 列表不能像数组一样转置
    • 列表没有维度的概念

与上文中的copy()和deepcopy()的概念类似：

用array得出的数组会独立出来一个数组，改变原来的值，新值不会变。(类似于deepcopy())但是用asarray得出来的新数组会随着原数组的改变而改变（类似于copy()）

7、P64 np.random.shuffle()的作用是将9个类的数据打乱顺序（假如第一个类中有50个样本，第二类中有100个样本等等….shuffle()之后第一个类中会随机出现50个样本，第二个类里面随机出现100个样本—>类似洗牌）所以这样准确率是19%，而不是我之前错想的那种1/46

8、在回归问题中常常涉及到标准化，在这里对训练集和测试集进行标准化处理都需要用训练集得出来的平均值和标准差。切记不能用测试集得出平均值和标准差再给测试集标准化。

10、python中的concatenate()函数的作用是将函数中的两个东西连接组装起来，但是前提是这两个部分的形状相同。其中axis=0表示操作行数据，每一行每一行的操作。axis = 1 表示操作列数据，每一列每一列的操作。默认是axis = 0
