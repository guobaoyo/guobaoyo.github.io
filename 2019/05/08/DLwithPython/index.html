本笔记是基于弗朗索瓦•肖莱(François Chollet)的DeepLearning with Python书籍的课程笔记。

在看书中曾遇到的不会的点：

1、在P22有个准备图像数据

  • astype的作用是将数据类型转换为后面（）部分的数据类型，在这里是float类型。

  • 后面除以255是因为灰度图的范围是[0,255]，然后再除以255，这样每个值的范围都集中在[0,1]，便于计算。

  • to_categorical()函数的作用是将类别向量转化为二进制（0和1）的矩阵类型表示。

2、epoch 和iteration 和 batchsize的区别（假设整个训练集为N）

  • batchsize可大可小，如果大到N，那么这叫full batch learning

  • batchsize如果小到1，就是每次只训练一个样本，就叫做online learning

  • 所以batchsize一般会取中间值n

    iteration =N/n

    epoch：整个N个训练集全部训练过一遍后epoch增加1 ，一般不会将epoch设置为1，因为一个大型的神经网络不可能将训练集通过一遍就训练完毕，所以随着epoch的增加，模型将从欠拟合逐步转化到过拟合。

书中写的是“避免覆盖输入张量”

但是我查copy()和deepcopy()的区别：https://blog.csdn.net/u010712012/article/details/79754132 里面讲copy()是浅复制，改变原来的值那么拷贝出来的值也会改变（在上面的文章中有标签的概念讲的很清晰可以参考）。deepcopy()是深复制，拷贝出来的东西与原来的是独立的。此外我又在电脑上试了一下，在整体改变x1的值之后必须再copy一遍才会生效，详见贴图：

书中写的“避免覆盖输入张量”我觉着应该用deepcopy()更合适，这样会独立出来一个对象对其进行修改并不会对原来的x造成影响。这样才会达到注释写的“避免覆盖”的作用。
