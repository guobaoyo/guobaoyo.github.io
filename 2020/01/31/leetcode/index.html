    1.两数之和

    class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashmap = {}
        for index,num in enumerate(nums):
            another_num = target - num
            if another_num in hashmap:
                return [hashmap[another_num],index]
            hashmap[num] = index
        return None

    上面是看到了一个特别省时省力的解法，存疑在于如果给定的是[1,2,3,6,7]，target是9的话，合理的输出应该是什么？（自己写了个答案，挺丑的）

    def twoSum(nums, target):
    hashmap = {}
    hashmap1 = {}
    hashmap3 = {}
    for index, num in enumerate(nums):
        another_num = target - num
        if another_num in hashmap:
            hashmap1[num] = index
            hashmap1[another_num] = hashmap[another_num]
        hashmap[num] = index
    j = 0
    for i in hashmap1:
        hashmap3[j] = hashmap1[i]
        j += 1
    for i in hashmap3:
        if (i%2==0):
            print('['+str(hashmap3[i])+','+str(hashmap3[i+1])+']')
        else:
            continue

def main():
    a = [1,6,2,3,7]
    target = 9
    print(twoSum(a,target))

if __name__ == '__main__':
    main()

    2.买卖股票的最佳时机

    参考https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock/solution/yi-ge-fang-fa-tuan-mie-6-dao-gu-piao-wen-ti-by-l-3/

    第一题与第二题的区别点在于k的次数的限制，第一题是k=1第二题k不限制，所以第一题中当天若持有股票就只能是当天买入的，且买入的前一天的dp[i-1][0]为0所以在代码中可以将其省略掉：

    class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n_day = len(prices)#表示第几天
        if n_day == 0:
            return 0
        rest = 2
        dp = [None]*n_day
        for i in range(len(dp)):
            dp[i] = [0]*rest
        for i in range(n_day):
            dp[i][0] = 0            
        for i in range(n_day):
            if(i==0):
                dp[i][0] = 0
                dp[i][1] = -prices[i]
            else:
                dp[i][0] = max(dp[i-1][0],dp[i-1][1]+prices[i])
                dp[i][1] = max(dp[i-1][1],-prices[i])

        return dp[n_day-1][0]

    720. 词典中最长的单词

    看评论去有位大神的做法比较特别，记录备忘，python中sort()的用法，

    words = ["a","banana","app","appl","ap","apply","apple"]
words.sort()
print(words)
words.sort(key = len,reverse = True)
print(words)

    第一个sort()首先按照字典序排列，其次按照长度从小到大。

    [‘a’, ‘ap’, ‘app’, ‘appl’, ‘apple’, ‘apply’, ‘ba’, ‘banana’]

    第二个sort()按照长度从大到小的顺序：

    [‘banana’, ‘apple’, ‘apply’, ‘appl’, ‘app’, ‘ap’, ‘ba’, ‘a’]

    注意在本题中先使用第一种方法将apple放到apply前面，再第二次排序，将长度较长的放在前面

    205. 同构字符串

    判断同构字符的时候，用到

    map(s.index, s)

    str 类型数据拥有内置函数 index，输入一个子字符串，可以返回子字符串在 str 中第一次出现的索引，若不存在则报错：
    可以用

    list(map(s.index, s))

    也可以像290题一样用

    [*map(s.index, s)]
