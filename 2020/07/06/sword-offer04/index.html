    解法

    出自初学者和Krahets

    简单粗暴直接比较

    class Solution:
    def findNumberIn2DArray(self, matrix: List[List[int]], target: int) -> bool:
        for i in matrix:
            for j in i:
                if target == j:
                    return True
        return False

    最无脑的方法没有用到数组matrix本身的递增属性，可以作为最开始的基础想法方便后面改进

    从右上角往左下角看👍

    class Solution:
    def findNumberIn2DArray(self, matrix: List[List[int]], target: int) -> bool:
        if len(matrix)==0:return False
        i,j = 0,len(matrix[0])-1
        while i<=len(matrix)-1 and j >=0:
            if matrix[i][j]>target:
                j-=1
            elif matrix[i][j]<target:
                i+=1
            else:return True
        return False

    二分法

    如果将矩阵展开是一个递增的列表，那么就可以使用如下解法

    仿照https://leetcode-cn.com/problems/search-a-2d-matrix/solution/ju-zhen-de-er-fen-cha-zhao-shou-hui-man-hua-tu-jie/ 写的python版本的

    class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if len(matrix)==0 or len(matrix[0])==0:
            return False
        m = len(matrix)
        n = len(matrix[0])
        left = 0
        right = m*n - 1
        mid = (left+right)//2
        while left<right:
            mid = (left+right)//2
            # print('*')
            # print(left)
            # print(right)
            # print(mid)
            # print('*')
            if matrix[mid//n][mid%n]>target:
                # print('大了')
                right = mid
            elif matrix[mid//n][mid%n]<target:
                # print('小了')
                left = mid + 1
            else:
                left = mid
                break
        #print(left)
        if matrix[left//n][left%n]==target:
            return True
        return False
